# شارك معنا

تطبيق صفّي بسيط مبني باستخدام HTML وCSS وJavaScript وFirebase Realtime Database.

## بيانات الدخول

- المعلم: `tech`
- الطالب: `STD`

هذه كلمات مرور تدريبية موجودة داخل كود الواجهة وليست حماية مناسبة لتطبيق حقيقي.

## طريقة الاستخدام

### المعلم

1. يسجل الدخول بصفة معلم.
2. يرفع ملف Excel يحتوي أسماء الطلبة في العمود الأول من الورقة الأولى.
3. يكتب السؤال بخط واضح في مربع السؤال.
4. يضغط `بدء مشاركة جديدة`.
5. يتابع إجابات الطلبة وعدد مشاركات كل طالب.
6. يستطيع وضع `إعجاب` أو `غير صحيح` على أي إجابة.

بدء مشاركة جديدة يمسح إجابات الجولة السابقة، لكنه يحتفظ بعدد المشاركات التراكمي لكل طالب.

### الطالب

1. يسجل الدخول بصفة طالب.
2. يختار اسمه من القائمة.
3. يكتب إجابته ويرسلها.
4. يمكنه تحديث إجابته في نفس الجولة.
5. يستطيع مشاهدة إجابات بقية الطلبة وتقييم المعلم.

## تنسيق ملف Excel

ضع عنوانًا اختياريًا مثل `اسم الطالب` في الخلية الأولى، ثم ضع اسمًا واحدًا في كل صف:

```text
اسم الطالب
محمد أحمد
سارة علي
محمود حسن
```

يدعم التطبيق ملفات `.xlsx` و`.xls` و`.csv`.

## قاعدة البيانات

يستخدم التطبيق المسارات التالية:

- `students`: أسماء الطلبة وعدد المشاركات.
- `activity`: السؤال الحالي ووقت بدء الجولة.
- `responses`: إجابات الجولة الحالية وتقييم المعلم.

قواعد القراءة والكتابة المفتوحة مناسبة للتدريب فقط. في تطبيق حقيقي يجب استخدام Firebase Authentication وقواعد أمان مناسبة.
